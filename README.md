# ConTags

A sleek, modern, and fully secure client-side web application for searching and filtering through your exported contacts via custom tags.

## ✨ Features
- **100% Client-Side Privacy:** Your contacts never leave your device. The CSV parsing (using `PapaParse`) happens entirely within your browser memory.
- **Cross-Platform PWA:** Install ConTags directly to your Android or Desktop home screen. It feels and runs exactly like a native app.
- **Instant Search & Filtering:** Quickly search by name, phone, or email. Filter contacts by checking off required label tags.
- **Premium UI:** "Clean minimal white mode" aesthetic with smooth micro-animations and a responsive design.

## 🚀 How to Use
1. Export your contacts as a standard Google Contacts CSV format.
2. Go to the hosted web application.
3. Click **Load CSV** and select your file.
4. Use the sidebar checkboxes or the top search bar to filter your address book.

## 🛠️ Tech Stack
This application is built using standard, framework-free web technologies:
- HTML5
- Vanilla CSS 
- Vanilla JavaScript
- [PapaParse](https://www.papaparse.com/) (For CSV Data Extraction)
- [Phosphor Icons](https://phosphoricons.com/)

## 📱 Mobile Installation
Because ConTags is a Progressive Web App (PWA):
1. Navigate to the app link using Chrome on your Android device.
2. Tap the browser menu (⋮).
3. Select **"Add to Home Screen"**.
4. The app will install onto your device like a regular application.
